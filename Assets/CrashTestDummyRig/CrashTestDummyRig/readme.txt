#		- Mirror/Flip poses
#		- Pin Controls one to another

#=================================================================================================================================================
#=================================================================================================================================================
#	RIG INTERFACE INSTALLATION
#=================================================================================================================================================
#=================================================================================================================================================

1.- Copy the contents of the 'scripts' folder into Maya's script folder 
	Windows: 'C:\Users\[yourusername]\Documents\maya\[yourmayaversion]\scripts\'
	Linux:  '/home/[yourusername]/maya/[yourmayaversion]/scripts/'

2.- Copy the contents of the 'shelves' folder into Maya's shelves folder 
	Windows: 'C:\Users\[yourusername]\Documents\maya\[yourmayaversion]\prefs\shelves\'
	Linux:  '/home/[yourusername]/maya/[yourmayaversion]/prefs/shelves/'

3.- You can copy the rigs from the 'mayafiles' folder into one of Maya's projects folder or open the files directly.

4.- Enjoy!
